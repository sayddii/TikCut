## ✂ Что это такое

TikCut — десктопная программа с графическим интерфейсом для нарезки видео на короткие части под TikTok. Загружаешь серию или фильм, настраиваешь длину частей, нажимаешь кнопку — программа сама всё режет и называет файлы как надо.

Никакой регистрации, никаких облаков, никаких подписок. Работает полностью на твоём компьютере через [ffmpeg](https://ffmpeg.org).

---

## ⚡ Возможности

| Функция | Описание |
|---|---|
| **✂ Нарезка на части** | Режет видео на части по 1–3 минуты. Файлы называются `Серия 1 - Часть 1 из 6.mp4` |
| **🎬 Разделение серий** | В одном файле две серии? Указываешь метку времени — программа сначала разделит, потом нарежет каждую |
| **🕐 Вырезание заставки** | Убирает интро по времени (например первые 30 секунд) или автоматически через анализ кадров |
| **⚡ Быстрый режим** | Копирует поток без перекодирования — часовое видео нарезается за секунды |
| **🎯 Точный режим** | Перекодирует через x264 — срез точно по секунде, нужен когда важна точность |
| **📋 Подписи для TikTok** | После нарезки в логе появляются готовые подписи для каждой части — копируй и вставляй |

## 🚀 Установка и запуск

### 1. Установи Python

Скачай с [python.org/downloads](https://python.org/downloads).

> ⚠️ При установке на Windows обязательно поставь галочку **"Add Python to PATH"**

### 2. Установи ffmpeg

**Windows:**
```bash
winget install ffmpeg
```
Или скачай вручную с [ffmpeg.org](https://ffmpeg.org/download.html) и добавь папку `bin` в PATH.

**macOS:**
```bash
brew install ffmpeg
```

**Linux (Ubuntu/Debian):**
```bash
sudo apt install ffmpeg
```

### 3. Скачай TikCut

```bash
git clone https://github.com/ВАШ_НИК/tikcut.git
cd tikcut
```

Или просто скачай `tiktok_splitter_gui.py` напрямую.

### 4. Запусти

```bash
python tiktok_splitter_gui.py
```

На macOS/Linux:
```bash
python3 tiktok_splitter_gui.py
```

Никаких дополнительных библиотек устанавливать не нужно — только стандартный `tkinter` который идёт с Python.

## 🎬 Как пользоваться

### Нарезка одного видео

1. Открой вкладку **✂ Нарезка**
2. Нажми **Выбрать...** и укажи видеофайл
3. Название серии подставится автоматически из имени файла — можешь изменить
4. Выбери длину каждой части ползунком (1:00 — 3:00)
5. Опционально: включи **"Пропускать заставку"** и укажи сколько секунд вырезать
6. Нажми **▶ НАРЕЗАТЬ ВИДЕО**

Готовые файлы появятся в папке `tiktok_parts` рядом со скриптом (или в той которую укажешь).

### Разделение двух серий из одного файла

1. Открой вкладку **🎬 Разделить серии**
2. Выбери видеофайл — программа покажет его длительность
3. Укажи название каждой серии и **когда заканчивается** первая серия
4. Формат времени: `01:22:45` или `82:45` или просто `4965` (секунды)
5. Последняя серия идёт до конца файла — её метку писать не надо
6. Добавь сколько угодно серий кнопкой **+ Добавить серию**
7. Нажми **▶ РАЗДЕЛИТЬ И НАРЕЗАТЬ**

Результат:
```
series_parts/
├── 01_series/
│   ├── 01. Серия 1.mp4
│   └── 02. Серия 2.mp4
└── 02_tiktok_parts/
    ├── Серия 1/
    │   ├── Серия 1 - Часть 1 из 5.mp4
    │   └── Серия 1 - Часть 2 из 5.mp4
    └── Серия 2/
        └── ...
```

## ❓ FAQ

**Чем отличается Быстрый режим от Точного?**

- **Быстрый** — копирует видео без перекодирования. Нарезка занимает секунды. Срез идёт по ближайшему ключевому кадру — может отличаться от указанного времени на 1–5 секунд. Для TikTok этого более чем достаточно.
- **Точный** — перекодирует через x264. Работает в 10–50 раз медленнее. Срез идёт точно по указанной секунде. Нужен когда важно попасть в конкретный момент.

**Программа говорит "ffmpeg не найден"**

После установки ffmpeg перезапусти CMD/Terminal — PATH обновляется только при новом сеансе.

**Авто-определение заставки не работает**

Авто-детект анализирует яркость кадров и работает хорошо для заставок с однородным фоном (чёрный экран, статичный логотип). Для сложных заставок лучше использовать режим "По времени" и вписать длину вручную.

**Можно ли публиковать прямо из программы?**

Нет — TikTok не предоставляет публичный API для загрузки видео обычным пользователям. Загружать нужно вручную через приложение или сайт.
